# smart-binary-options-trader

A Reinforcement learning agent for event -based crypto trading

**Episode**
One full trading day (9:00 - 24:00)

**Time step**
One hourly contract

## Starting project

`uv run main.py`

# Dataset

Used a public available bitcoin data at 1 minute interval dataset [Link](https://www.kaggle.com/datasets/mczielinski/bitcoin-historical-data)
